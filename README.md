# Simple Ditther

A web-based application for applying dithering and ASCII effects to images and videos.

## Features
- **Dithering**: Apply various algorithms like Floyd-Steinberg, Atkinson, etc.
- **ASCII Art**: Convert images and videos to ASCII text.
- **Video Support**: Process video files in real-time and export them.
- **Customization**: Adjust point size, palettes, brightness, contrast, and more.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kosy78/simple-ditther.git
   cd simple-ditther
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```
